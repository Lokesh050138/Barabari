Mission Statement 2024

We are a section 8 tech education non-profit working towards

1. Creating 500+ freelancing opportunites for students from rural low-income families.

2. Mentoring 500+ students from marginalized sector employed in high quality tech sector roles currently accesible to select universities.

3. Supporting 100+ affordable schools set up computer education infrastructure and complete our digital literacy modules

4. Fund 10+ candidates from marginalized communities set up their tech entreprenaurial venture.

Tech Rozgaar Program

1. Candidates from marginalized communities get mentored esteemed tech professional toward their first tech job/intership

2. We help candidates build a strong tech foundation, build profecient communication skills, create an elaborate work profile and network with industry leaders.

3. Our core focus is on remote jobs and freelancing opportunites that help candidates from rural low-income families join the tech workforce smoothly

Digital Literacy Program

1. Sopporting government schools in setting up computer education infrastructure, recruit and train faculty and build a tech education curriculum

2. Build a strong tech foundation in students, teach them to identify problems around them, dissect and come up with solutions and implement these solutions using tech tools.

3. Project based learning to make marginalized communities comfortable with tech
